# Day 3 - MySQL Fundamentals

Today I learned the basics of MySQL and database management. I installed MySQL and MySQL Workbench, created a database, designed tables using appropriate data types and primary keys, and inserted sample records. This helped me understand how data is stored and organized in a relational database.

I also practiced essential SQL operations such as INSERT, SELECT, WHERE, UPDATE, and DELETE. Using a simple device management database, I performed data retrieval, filtering, modification, and deletion queries to gain hands-on experience with database operations.
